# Nested class와 Inner class에 대해 알아보자

---

## 🧐 우선 Nested class가 뭘까?

### Nested class?

Nested class는 클래스 내에 클래스를 선언한 방식으로, 중첩 클래스를 의미함

중첩 클래스는 `non-static nested class`인 `inner class`와 `static nested class`로 나뉨

### 중첩 클래스의 장점

- 외부 클래스의 쉬운 참조
- 캡슐화 - 외부에 불필요한 코드를 감춤으로써 코드의 복잡성을 줄일 수 있음
